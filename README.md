# TechCity 
**E-Commerce Mobile App for Electronics**

Flutter mobile application for buying and selling electronic goods 
with role-based access for Admin and Users.

## Tech Stack
- **Framework**: Flutter (Dart)
- **Backend & Database**: Supabase
- **Platform**: Android

## Features
### User
- Browse & explore electronics catalog
- Product detail & reviews
- Shopping cart & checkout
- Top-up balance & payment
- Order history
- Edit profile & settings

### Admin
- Add, edit, delete products
- Manage order status
- View profit & analytics
